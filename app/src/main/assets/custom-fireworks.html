Loading Custom Fireworks in iFireworks

iFireworks uses JSON to store and reference information about firework items. You can either use the provided Google Sheets template or you can write the JSON yourself.

To use the template click the link to make a copy of the template then you can add your fireworks

Get the Template

When using the template download as CSV and then use CSV to JSON to convert it to allow it to be read by the app. Also make sure to remove all the rows you haven't used to prevent the app from creating empty items.

Key Data Type Description
ID integer Numerical ID assigned to the firework item
Name string Name of the item
SKU string SKU of the item
Category string "500G Cake", "200G Cake", "Novelties", "NOAB", "Rockets", "Saturn Missile Battery" , "Girandola", "Shells", or "Roman Candles" (case sensitive)
Duration string A numerical or "varries" if it is an assortment
Shots integer Number of shots of an item or 0 if the item does not have shots such as a sparkler.
imgurl string URL of an image of the product
demo string embed url of the product (https://www.youtube.com/embed/0T07_ArQFs0)
Performance string Item performance

Any value except ints that does not have a value should not be left blank rather null should be entered
